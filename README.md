## DS 6600 Final Project

This repository contains all the code for the final project. Data manipulation can be found in `get_data.ipynb`, with an ER diagram describing the final tables documented at https://dbdocs.io/vdq8tp/DS6600-Project. The dashboard can be hosted locally with Docker, or can be found online at https://jguo.pythonanywhere.com/dashboard. The final report can be found in the `reports` folder.

### Gen AI Statement

Generative AI was used to assist with regex conditions when cleaning MSA names in ACS data. No other code was generated with AI. None of the written information present in the reports were generated with AI.
